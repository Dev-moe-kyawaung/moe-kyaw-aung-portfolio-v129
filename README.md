## 🎨 Customization

### **1. Update Personal Information**
```html
<h1 class="hero-title">⭐ YOUR NAME</h1>
<p class="hero-subtitle">YOUR TITLE</p>
```

### **2. Change Color Scheme**
```css
:root {
    --primary-dark: #0a0a0a;           /* Background */
    --accent-cyan: #00d4ff;            /* Primary accent */
    --accent-pink: #ff006e;            /* Secondary accent */
    --accent-gold: #ffd60a;            /* Tertiary accent */
}
```

### **3. Update Social Links**
```html
<a href="https://github.com/YOUR_USERNAME" target="_blank">
    <i class="fab fa-github"></i>
</a>
```

### **4. Modify Featured Projects**
```html
<div class="app-card fade-up">
    <div class="app-icon">🎯</div>
    <div class="app-name">Your Project</div>
    <div class="app-description">Description</div>
    <a href="https://github.com/link">View on GitHub →</a>
</div>
```
