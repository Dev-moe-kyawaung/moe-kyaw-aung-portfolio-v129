# 🚀 Quick Start Guide

Get your portfolio website up and running in minutes!

---

## 📥 Installation

### **Method 1: Direct Download (Fastest)**
```bash
1. Download moe-kyaw-aung-portfolio-v129.html
2. Double-click to open in browser
3. Done! 🎉
```

### **Method 2: GitHub Pages (Recommended)**
```bash
# Clone repository
git clone https://github.com/Dev-moe-kyawaung/moe-kyaw-aung-portfolio-v129.git
cd moe-kyaw-aung-portfolio-v129

# Push to GitHub Pages
# In repo settings, enable GitHub Pages
# Select main branch as source
```

### **Method 3: Local Server (Development)**
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Open: http://localhost:8000
```

## ✏️ Customization

### **Step 1: Edit Your Name**
Find and replace in HTML:
```html
<!-- Before -->
<h1 class="hero-title">⭐ MOE KYAW AUNG</h1>

<!-- After -->
<h1 class="hero-title">⭐ YOUR NAME</h1>
```

### **Step 2: Update Colors (Optional)**
```css
:root {
    --accent-cyan: #00d4ff;    /* Change primary color */
    --accent-pink: #ff006e;    /* Change secondary color */
    --accent-gold: #ffd60a;    /* Change accent color */
}
```

### **Step 3: Add Your Links**
Replace URLs in social icons:
```html
<a href="https://github.com/YOUR_USERNAME" target="_blank">
    <i class="fab fa-github"></i>
</a>
```

### **Step 4: Update Contact Info**
```html
<a href="tel:+95988900000">Your Phone</a>
<a href="mailto:your@email.com">Your Email</a>
```

### **Step 5: Add Your Projects**
```html
<div class="app-card">
    <div class="app-icon">🎯</div>
    <div class="app-name">Your Project</div>
    <div class="app-description">Description</div>
    <a href="https://github.com/link">View on GitHub →</a>
</div>
```

## 🚢 Deployment

### **Option 1: GitHub Pages**
```bash
1. Push HTML to repository
2. Go to Settings → Pages
3. Select main branch as source
4. Save
5. Access at: username.github.io/repo-name
```

### **Option 2: Netlify**
```bash
1. Connect GitHub repository
2. Deploy button → Done!
3. Custom domain (optional)
```

### **Option 3: Vercel**
```bash
1. Import GitHub repository
2. Deploy → Done!
3. Analytics included
```

### **Option 4: Traditional Hosting**
```bash
1. FTP/SFTP to server
2. Upload moe-kyaw-aung-portfolio-v129.html
3. Rename to index.html
4. Access via domain
```

## 🔧 Troubleshooting

### **Content Not Showing?**
- Check HTML file is in same directory
- Verify file name matches
- Clear browser cache (Ctrl+Shift+Delete)

### **Styles Not Working?**
- Ensure HTML file is complete
- Check CSS is inline (not external)
- Clear cache and reload

### **Dark Mode Not Working?**
- Check localStorage is enabled
- Try different browser
- Clear browser cache

### **Animations Slow?**
- Close unused browser tabs
- Check device performance
- Try different browser

## ⚡ Performance Tips

### **Faster Loading**
- Use CDN for images
- Compress images (tinypng.com)
- Minify CSS/JS (optional)
- Enable gzip compression

### **Optimize Images**
```html
<!-- Before (large file) -->
<img src="image.jpg" alt="Description">

<!-- After (optimized) -->
<img src="image-optimized.jpg" alt="Description" loading="lazy">
```
